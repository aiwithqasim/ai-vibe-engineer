# Kanban MVP

Next.js MVP in [`frontend`](d:\github\ai-vibe-coder\kanban\frontend) with one board, five renameable columns, drag-and-drop cards, add/delete cards, and no persistence.

## Run

```bash
cd frontend
npm install
npm run dev
```

## Test

```bash
cd frontend
npm run lint
npm run test
npm run test:e2e
```
